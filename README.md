**Professional backend development plugin for Claude Code** with 7 specialized agents and 7 skills covering programming fundamentals, databases, APIs, architecture, caching, DevOps, and security.

## Features

- **7 Specialized Agents** for comprehensive backend development
- **7 Production-Ready Skills** with Golden Format structure
- **SASMP v1.3.0 Compliant** with full agent-skill bonding
- **Real Templates & Scripts** for immediate use
- **Comprehensive Documentation** for all components

---

## Agents

| Agent | Specialization | Bonded Skills |
|-------|---------------|---------------|
| `programming-fundamentals-agent` | Languages, Git, Package Management | languages |
| `database-management-agent` | SQL, NoSQL, Query Optimization | databases |
| `api-development-agent` | REST, GraphQL, Authentication | api-design |
| `architecture-patterns-agent` | SOLID, Design Patterns, Microservices | architecture |
| `caching-performance-agent` | Redis, Load Balancing, Monitoring | performance |
| `devops-infrastructure-agent` | Docker, Kubernetes, CI/CD | devops |
| `testing-security-agent` | OWASP, Testing, Compliance | security |

## Skills

| Skill | Description | Assets |
|-------|-------------|--------|
| `languages` | Programming language selection and setup | YAML, Shell |

## Usage

### Invoke an Agent

```
"Help me design a database schema for my e-commerce application"
→ database-management-agent activates

"Containerize my FastAPI application"
→ devops-infrastructure-agent activates

"Audit my app for OWASP Top 10 vulnerabilities"
→ testing-security-agent activates
```

### Use a Skill

```
"Use the api-design skill to create OpenAPI spec"
→ Generates OpenAPI 3.0 template

"Apply the performance skill for Redis caching"
→ Provides Redis configuration and patterns
```

## Security Notice

This plugin is designed for **authorized development use only**. Always follow security best practices:

- Never commit secrets or credentials
- Use environment variables for sensitive data
- Follow OWASP guidelines for secure development
- Regular security scanning with provided tools
