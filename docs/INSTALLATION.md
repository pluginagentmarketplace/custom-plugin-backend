# Installation Guide

## Prerequisites

- Claude Code CLI installed
- Git

## Installation Steps

### 1. Clone the Repository

```bash
git clone https://github.com/pluginagentmarketplace/custom-plugin-backend.git
```

### 2. Install via Claude Code

```bash
# Navigate to Claude Code plugins directory
cd ~/.claude/plugins

# Install the plugin
claude plugin install custom-plugin-backend
```

### 3. Verify Installation

```bash
# List installed plugins
claude plugin list

# Should show: backend-development-assistant v2.0.0
```
